Glow & Beauty - E-commerce de Cosméticos 💄✨

Glow & Beauty es una plataforma de comercio electrónico desarrollada como una solución integral para la venta de productos de maquillaje y belleza. El sistema permite gestionar el ciclo completo de venta, desde la navegación del catálogo hasta la gestión administrativa de pedidos mediante integración directa con WhatsApp.
🚀 Características Principales
🛒 Tienda Pública (Frontend)

    Catálogo Dinámico: Navegación por categorías y marcas.

    Carrito de Compras: Sistema de gestión de productos en sesión con persistencia durante la navegación.

    Checkout Optimizado: Formulario de finalización de compra con validación de datos y captura de información de contacto (WhatsApp).

    Interfaz Responsive: Diseño elegante y adaptable enfocado en la experiencia de usuario (UX).

⚙️ Panel de Administración (Backend)

    Gestión de Órdenes: Visualización detallada de pedidos entrantes.

    Integración con WhatsApp: Acceso directo desde el panel para contactar al cliente con mensajes predefinidos personalizados (Nombre, ID de pedido).

    Control de Inventario: CRUD completo para productos, categorías y marcas.

    Seguridad de Datos: Implementación de transacciones SQL para asegurar la integridad entre las tablas de órdenes y detalles.

🛠️ Tecnologías Utilizadas

    Lenguaje: PHP 8.1+ (Programación orientada a objetos y procedimientos).

    Base de Datos: MySQL (Gestionado mediante MySQL Workbench).

    Conectividad: PDO (PHP Data Objects) para consultas preparadas y prevención de SQL Injection.

    Frontend: HTML5, CSS3 (Custom styling con variables modernas) y JavaScript.

    Iconografía: Font Awesome 6.

    Tipografías: Google Fonts (Montserrat).
